Мої навики

HTML 5 - HyperText Markup Language

Cтандартна мова розмітки веб-сторінок в Інтернеті. Верстаю за стандартам W3C (валідний код). Код проходить перевірку на validator.w3.org.

CSS 3 - Cascading Style Sheets

Каскадні таблиці стилів. CSS використовують для візуальної презентації сторінок, написаних HTML.

Java Script

Мова програмування. Дає можливість користувачу взаємодіяти з браузером. Займаюсь написанням простих скриптів для сторінок із використанням JS бібліотек.

J Queri

jQuery є найпопулярнішою бібліотекою JavaScript, яка посилено використовується на сьогоднішніій день. JQuery використовується понад половиною від мільйона найвідвідуваніших сайтів. Дозволяє створювати анімацію для ефектів високого рівня. Це сприяє створенню динамічних веб-сторінок.

Responsive web design

Розробляю сторінки які правильно відображаються на екранах різних пристроїв і динамічно підлаштовуються під розмір вікна браузера.

Bootstrap

Bootstrap є найбільш популярним HTML, CSS, та JS фреймворком для зовнішнього вигляду та інтерактивності веб сторінок.

SASS

Інтерпретується в каскадні таблиці стилів (CSS).Sass призначений для підвищення рівня абстракції коду та спрощення файлів CSS.

Gulp

Gulp - менеджер задач, орієнтованих на frontend - розробку, по суті - система управління усіма задачами
